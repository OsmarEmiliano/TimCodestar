1)Agrega los archivos modificados:
Abre tu terminal y navega hasta el directorio de tu proyecto. Utiliza el siguiente comando para agregar 
todos los archivos modificados al área de preparación de Git.
 
---- git add . ---

2)Confirma los cambios:
Después de agregar los archivos, realiza una confirmación
con un mensaje que describa tus cambios.

---- git commit -m "Index Semilla" ----

3)Sube los cambios al repositorio remoto (GitHub):
Utiliza el siguiente comando para subir tus cambios al repositorio en GitHub.

--- git push -u origin  main ----
